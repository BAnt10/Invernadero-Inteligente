Invernadero Inteligente

Objetivo
El objetivo principal de este proyecto es desarrollar un invernadero inteligente que permita monitorear y controlar las condiciones ambientales de manera automatizada para garantizar un entorno óptimo para el crecimiento de las plantas. Este sistema utilizará sensores de temperatura, humedad y luz para regular automáticamente el riego mediante una electroválvula, controlar la temperatura ambiental con un ventilador y monitorear los niveles de luz recibida por las plantas. Además, todos los datos recolectados se visualizarán en una interfaz diseñada en LabVIEW, facilitando la gestión y optimización del cultivo en tiempo real.
